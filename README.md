# bug-tracker
An app to keep track of bugs, issues and proposals of some app.

## Languages and technologies used
### Backend:
Java 17; Spring Boot, Spring Jpa, Spring Security; MySQL; Maven
### Frontend:
TypeScript; JavaScript; HTML; CSS; React; Vite; Npm
